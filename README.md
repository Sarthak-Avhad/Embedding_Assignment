# 🔤 Word Embeddings – NLP Assignment Project

## 📌 Project Overview

This project focuses on understanding and implementing **Word Embeddings** in Natural Language Processing (NLP).  
Word embeddings convert textual data into meaningful numerical vectors, capturing semantic relationships between words.

The notebook demonstrates how words can be represented in vector space and how these representations improve NLP model performance.

---

## 🎯 Objectives

- Understand the concept of word embeddings  
- Convert text data into numerical vectors  
- Explore semantic relationships between words  
- Apply embeddings in NLP tasks
